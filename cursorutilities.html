Cursor Utilities

The "Cursor Utilities" section provides functionality to manage and control the cursor in Unity applications.

SetCursorLocked()

Summary:

This function sets the cursor lock state to "Locked" in Unity, preventing it from leaving the game window.

Example usage:

void Start() { SetCursorLocked(); }

SetCursorConfined()

Summary:

This function sets the cursor lock state to "Confined" in Unity, preventing it from being locked in the game window.

Example usage:

void Start() { SetCursorConfined(); }

SetCursorUnlocked()

Summary:

This function sets the cursor lock state to "Unlocked" in Unity, preventing it from being locked in the game window.

Example usage:

void Start() { SetCursorUnlocked(); }

SetCursorVisible()

Summary:

This function sets the cursor lock state to "Unlocked" in Unity, preventing it from being locked in the game window.

Example usage:

void Start() { SetCursorVisible(); }

SetCursorInvisible()

Summary:

This function sets the cursor lock state to "Unlocked" in Unity, preventing it from being locked in the game window.

Example usage:

void Start() { SetCursorInvisible(); }
