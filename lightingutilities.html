Advanced API Docs Help

Lighting Utilities

ToggleLight(GameObject obj)

Summary:

Toggles the visibility of a Light component on the given GameObject.

Parameters:

obj = The GameObject with a Light component to be toggled.

Example usage:

Start() { GameObject myLightObject; LightingUtilities.ToggleLight(myLightObject); }

ChangeLightIntensity(GameObject obj, float newIntensity)

Summary:

Changes the intensity of a Light component on the given GameObject to the specified value.

Parameters:

obj = The GameObject with a Light component to be modified.

newIntensity = The new intensity value for the Light component.

ChangeLightColor(GameObject obj, Color newColor)

Summary:

Changes the color of a Light component on the given GameObject to the specified color.

Parameters:

obj = The GameObject with a Light component to be modified.

newColor =The new color for the Light component.

MoveLightToPosition(GameObject obj, Vector3 newPosition)

Summary:

Moves the Light component on the given GameObject to the specified position in the scene.

Parameters:

obj = The GameObject with a Light component to be moved.

newPosition = The new position for the Light component in the scene.

FadeLightIntensity(Light light, float targetIntensity, float duration)

Summary:

Fades a Light component's intensity to a target value over a specified duration.

Parameters:

light = The Light component to be faded. targetIntensity = The target intensity to reach. duration = The duration of the fade in seconds.

FlashLight(Light light, float frequency)

Summary:

Flashes a Light component on and off with a specified frequency.

Parameters:

light = The Light component to flash.

frequency = The flashing frequency in flashes per second.

SetLightRange(Light light, float newRange)

Summary:

Sets the range of a Light component to cover a specified radius.

Parameters:

light = The Light component to adjust.

newRange = The new range for the Light component.

CreateLight(Vector3 position, Color color, float intensity, float range)

Summary:

Creates a new GameObject with a Light component and positions it in the scene.

Parameters:

position = The position in the scene where the new light should be placed.

color = The color of the new light.

intensity = The intensity of the new light.

range = The range of the new light.

returns = The GameObject with the created Light component.

RotateSpotlight(Light light, float rotationSpeed)

Summary:

Rotates a spotlight-like Light component around its pivot point.

Parameters:

light = The Light component to be rotated.

rotationSpeed = The speed of rotation in degrees per second.

FlickerLight(Light light, float minIntensity, float maxIntensity)

Summary:

Flickers a Light component to simulate a candle-like effect.

Parameters::

light = The Light component to flicker.

minIntensity = The minimum intensity during flickering.

maxIntensity = The maximum intensity during flickering.

PulsateLightIntensity(Light light, float minIntensity, float maxIntensity, float pulsationSpeed)

Summary:

Summary:

Adjusts the spotlight cone angle of a Light component.

CreatePointLight(Vector3 position, Color color, float intensity, float range)

Summary:

Creates a point light GameObject at the specified position with given parameters.

Parameters:

position = The position in the scene where the point light will be created.

StrobeLight(Light light, float strobeFrequency)

Strobe a Light component at a specified frequency.

Parameters:
