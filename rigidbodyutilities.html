Rigidbody Utilities

SetVelocity3D(Rigidbody rb, float x, float y, float z)

Summary:

Set the velocity of a Rigidbody in 3D space.

Parameters:

rb = The Rigidbody for which to set the velocity.

x = The velocity along the X-axis.

y = The velocity along the Y-axis.

z = The velocity along the Z-axis.

SetVelocity2D(Rigidbody rb, float x, float y)

Summary:

Set the velocity of a Rigidbody in 2D space.

Parameters:

rb = The Rigidbody for which to set the velocity.

x = The velocity along the X-axis.

y = The velocity along the Y-axis.

Vector3 GetVelocity(this Rigidbody rb)

Summary:

Get the velocity of a Rigidbody as a Vector3.

Parameters:

rb = The Rigidbody for which to retrieve the velocity.

returns = The velocity of the Rigidbody as a Vector3.

GetVelocity2D(this Rigidbody2D rb2d)

Summary:

Get the velocity of a Rigidbody2D as a Vector2 in 2D space.

Parameters:

rb2d = The Rigidbody2D for which to retrieve the velocity.

returns = The velocity of the Rigidbody2D as a Vector2 in 2D space.

SetGravityScale(Rigidbody rb, float gravity)

Summary:

Set the gravity scale for a Rigidbody and enable gravity.

Parameters:

rb = The Rigidbody for which to set the gravity scale.

gravity =The gravity scale to set.

GetGravityScale(Rigidbody rb)

Summary:

Get the gravity scale for a Rigidbody.

Parameters:

rb = The Rigidbody from which to get the gravity scale.

returns = The gravity scale of the Rigidbody.

SetIsUsingGravity(Rigidbody rb, bool isUsingGravity)

Summary:

Set the useGravity state of a Rigidbody.

Parameters:

rb = The Rigidbody to modify.

isUsingGravity = The desired useGravity state (true to enable gravity, false to disable gravity).

SetIsKinematic(Rigidbody rb, bool isKinematic)

Summary:

Set the kinematic state of a Rigidbody.

Parameters:

rb = The Rigidbody to modify.

isKinematic = The desired kinematic state (true for kinematic, false for non-kinematic).

GetIsUsingGravity(Rigidbody rb)

Summary:

Get the gravity usage state of a Rigidbody.

Parameters:

rb = The Rigidbody to query.

returns = True if the Rigidbody is using gravity, false otherwise.

GetIsKinematic(Rigidbody rb)

Summary:

Get the kinematic state of a Rigidbody.

Parameters:

rb = The Rigidbody to query.

returns = True if the Rigidbody is kinematic, false otherwise.

ApplyForce(this Rigidbody rb, Vector3 force)

Summary:

Apply a force to the Rigidbody using the specified force vector.

Parameters:

rb = The Rigidbody to which the force will be applied.

force = The force vector to apply.

StopForce(this Rigidbody rb)

Summary:

Stop the Rigidbody by setting its velocity to zero.

Parameters:

rb = The Rigidbody to stop.

ApplyGravity(this Rigidbody rb)

Summary:

Allow a Rigidbody to apply gravity and simulate it by adding a force.

Parameters:

rb = The Rigidbody to modify.

DontApplyGravity(this Rigidbody rb)

Summary:

Prevent a Rigidbody from applying gravity.

Parameters:

rb = The Rigidbody to modify.

SetKinematic(this Rigidbody rb)

Summary:

Set a Rigidbody to be kinematic.

Parameters:

rb = The Rigidbody to modify.

SetNonKinematic(this Rigidbody rb)

Summary:

Set a Rigidbody to be non-kinematic.

Parameters:

rb = The Rigidbody to modify.

IsMoving(GameObject gameObject)

Summary:

Check if a GameObject is currently moving (has a non-zero velocity).

gameObject = The GameObject to check for movement.

Parameters:

True if the GameObject is moving, otherwise false.
