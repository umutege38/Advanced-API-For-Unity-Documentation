Transform Utilities

A utility class containing transform related functions.

GetTransform(this Transform transform)

Summary:

Gets the reference to the Transform component.

Parameters:

transform = The Transform component to get a reference to.

returns = The same Transform component.

Example usage:

void Start() { GameObject myObject = new GameObject("MyObject"); Transform myTransform = myObject.transform; Transform returnedTransform = TransformUtilities.GetTransform(myTransform); if (returnedTransform == myTransform) { Debug.Log("The returned Transform is the same as the original Transform."); } else { Debug.Log("Something went wrong; the Transforms are not the same."); } }

SetTransform(ref Transform transform, Transform value)

Summary: Sets the value of the provided Transform.

Parameter:

transform = Reference to the Transform to be set.

value = The Transform value to set.

Example usage:

void Start() { GameObject myObject = new GameObject("MyObject"); Transform myTransform = myObject.transform; GameObject anotherObject = new GameObject("AnotherObject"); Transform anotherTransform = anotherObject.transform; TransformUtilities.SetTransform(ref myTransform, anotherTransform); if (myTransform == anotherTransform) { Debug.Log("The value of myTransform has been set to anotherTransform."); } else { Debug.Log("Something went wrong; myTransform and anotherTransform are not the same."); } }

GetPosition(this Transform transform)

Summary:

Gets the position of the Transform in world space.

Parameters:

transform = The Transform component whose position is to be retrieved.

returns = The position of the Transform in world space.

Example usage:

void Start() { GameObject myObject = new GameObject("MyObject"); Transform myTransform = myObject.transform; myTransform.position = new Vector3(1.0f, 2.0f, 3.0f); Vector3 currentPosition = TransformUtilities.GetPosition(myTransform); Debug.Log("Current Position: " + currentPosition); }

GetRotation(this Transform transform)

Summary:

Gets the rotation of the Transform in world space.

Parameters:

transform = The Transform component whose rotation is to be retrieved.

returns = The rotation of the Transform in world space.

Example usage:

void Start() { GameObject myObject = new GameObject("MyObject"); Transform myTransform = myObject.transform; myTransform.rotation = Quaternion.Euler(45f, 30f, 60f); Quaternion currentRotation = TransformUtilities.GetRotation(); Debug.Log("Current Rotation: " + currentRotation); }

Example usage:

SetTransformValues(this Transform transform, Vector3 position, Quaternion rotation, Vector3 scale)

Summary:

Sets the position, rotation, and scale of the Transform in world space.

Parameters:

transform = The Transform component to be modified.

position = The new position in world space.

rotation = The new rotation in world space.

scale = The new scale in local space.

Example usage:

void Start() { GameObject myObject = new GameObject("MyObject"); Transform myTransform = myObject.transform; Vector3 newPosition = new Vector3(2.0f, 3.0f, 4.0f); Quaternion newRotation = Quaternion.Euler(45f, 30f, 60f); Vector3 newScale = new Vector3(1.5f, 2.0f, 0.5f); myTransform.SetTransformValues(newPosition, newRotation, newScale); Debug.Log("Updated Position: " + myTransform.position); Debug.Log("Updated Rotation: " + myTransform.rotation); Debug.Log("Updated Scale: " + myTransform.localScale); }

LookAt(Transform source, Transform target, float rotationSpeed)

Summary:

Makes the source Transform look at the target Transform smoothly.

Parameters:

source = The Transform that will perform the look-at operation.

target = The Transform to look at.

rotationSpeed = The speed at which the rotation should occur.

LookAtVertical(Transform source, Transform target, float rotationSpeed)

Summary:

Makes the source Transform look at the target Transform horizontally (yaw) while maintaining its vertical orientation.

Parameters:

source = The Transform that will perform the look-at operation.

target = The Transform to look at.

rotationSpeed = The speed at which the rotation should occur.

LookAtHorizontal(Transform source, Transform target, float rotationSpeed)

Summary:

Makes the source Transform look at the target Transform horizontally (yaw) while aligning their vertical positions.

Parameters:

source = The Transform that will perform the look-at operation.

target = The Transform to look at.

rotationSpeed = The speed at which the rotation should occur.
